## 项目依赖关系

```
Stardust (核心库) ←── Stardust.Extensions (ASP.NET Core扩展)
    ↑
Stardust.Data (数据层) ←── Stardust.Server (服务端API)
                                ↑
                          Stardust.Web (管理界面)
```

各独立工程：
- **StarGateway** → 引用 Stardust (SDK)，通过 API 与 StarServer 交互
- **StarAgent** → 引用 Stardust (SDK)，通过 API 与 StarServer 交互
- **DeployAgent** → 部署代理，通过 API 与 StarServer 交互

## 关键决策（跨模块）

| 决策 | 选择 | 说明 |
|------|------|------|
| 数据采集模式 | 客户端采样+统计，服务端汇总 | 降低 90%+ 网络传输和存储成本 |
| 统计计算 | 流式（5秒）+批计算（30秒）双模式 | 实时性 + 最终一致性 |
| 部署模式 | Standard/Shadow/Hosted/Task | 适应不同应用场景 |
| 数据清理 | Truncate/Delete 双模式 | 灵活适配数据库权限 |
| 通信协议 | HTTP + WebSocket 双通道 | 心跳/API 走 HTTP，命令走 WebSocket |
| 客户端接入 | AddStardust/UseStardust 一行代码接入 | 零改造集成

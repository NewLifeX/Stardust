应用跟踪器（AppTracer）

名称 显示名 类型 长度 精度 主键 允许空 备注
ID 编号 Int32 AI N
ProjectId 项目 Int32 N 资源归属的团队
AppId 应用 Int32 N
Name 名称 String 50 UQ N
DisplayName 显示名 String 50
Category 类别 String 50 同步自注册中心的应用分组，同时匹配告警分组，该应用未设置告警机器人时，采用告警分组的机器人设置
ItemCount 跟踪项 Int32 N 共有多少个埋点
Days 天数 Int32 N 共统计了多少天
Total 总次数 Int64 N 累计埋点采样次数
Enable 启用 Boolean N
Mode 跟踪模式 Int32 N 仅针对api类型，过滤被扫描的数据
Period 采样周期 Int32 N 单位秒
MaxSamples 正常数 Int32 N 最大正常采样数，采样周期内，最多只记录指定数量的正常事件，用于绘制依赖关系
MaxErrors 异常数 Int32 N 最大异常采样数，采样周期内，最多只记录指定数量的异常事件，默认10
EnableMeter 性能收集 Boolean N 收集应用性能信息，数量较大的客户端可以不必收集应用性能信息
WhiteList 白名单 String 200 要过滤Api操作名时的白名单，支持*模糊匹配如/Cube/*，支持^开头的正则表达式如^/Admin/
Excludes 排除项 String 200 要排除的操作名，支持*模糊匹配
Timeout 超时时间 Int32 N 超过该时间时强制采样，默认5000毫秒
MaxTagLength 最长标签 Int32 N 超过该长度时将截断，默认1024字符
RequestTagLength 请求标签长度 Int32 N HttpClient请求和WebApi请求响应作为数据标签的最大长度，小于0时不使用，默认1024字符
VipClients Vip客户端 String 50 高频次大样本采样，10秒100次，逗号分割，支持*模糊匹配
WebHook 钩子地址 String 500 监控数据转发给目标接口
AlarmThreshold 告警阈值 Int32 N 错误数达到该值时触发告警，0表示不启用，阈值和率值满足其一
AlarmErrorRate 告警错误率 Double N 错误率达到该值时触发告警，0表示不启用，阈值和率值满足其一
ItemAlarmThreshold 单项阈值 Int32 N 下级跟踪项错误数达到该值时触发告警，0表示不启用，阈值和率值必须同时满足
ItemAlarmErrorRate 单项错误率 Double N 下级跟踪项错误率达到该值时触发告警，0表示不启用，阈值和率值必须同时满足
AlarmRobot 告警机器人 String 500 钉钉、企业微信等
CreateUser 创建者 String 50
CreateUserID 创建者 Int32 N
CreateTime 创建时间 DateTime
CreateIP 创建地址 String 50
UpdateUser 更新者 String 50
UpdateUserID 更新人 Int32 N
UpdateTime 更新时间 DateTime
UpdateIP 更新地址 String 50


跟踪项（TraceItem）

名称 显示名 类型 长度 精度 主键 允许空 备注
Id 编号 Int32 AI N
AppId 应用 Int32 N
Kind 种类 String 50
Name 操作名 String 200 接口名或埋点名
DisplayName 显示名 String 50
Enable 启用 Boolean N
Rules 规则 String 200 支持多个埋点操作按照规则聚合成为一个跟踪项，用于处理多变的操作名，支持*模糊匹配，多个规则逗号隔开，多组规则分号隔开。如name=*/check*,*/ping*;clientId=10.10.*
Cloned 克隆 Boolean N 根据规则匹配，把跟踪数据克隆一份，形成另一个维度的统计数据
Timeout 超时时间 Int32 N 超过该时间时标记为异常，默认0表示不判断超时
Days 天数 Int32 N 共统计了多少天
Total 总次数 Int64 N 累计埋点采样次数
Errors 错误数 Int64 N
Cost 平均耗时 Int32 N 总耗时除以总次数，单位毫秒
AlarmThreshold 告警阈值 Int32 N 错误数达到该值时触发告警，0表示不启用，阈值和率值必须同时满足
AlarmErrorRate 告警错误率 Double N 错误率达到该值时触发告警，0表示不启用，阈值和率值必须同时满足
MaxRingRate 最大环比 Double N 环比昨日超过该率时触发告警，一般大于1，如1.2表示超20%，0表示不启用
MinRingRate 最小环比 Double N 环比昨日小于该率时触发告警，一般小于1，如0.7表示低30%，0表示不启用
AlarmGroup 告警组 String 50 使用告警组中指定的机器人
AlarmRobot 告警机器人 String 500 钉钉、企业微信等
TraceId 追踪 String 200 最新一次查看采样，可用于关联多个片段，建立依赖关系，随线程上下文、Http、Rpc传递
CreateIP 创建地址 String 50
CreateTime 创建时间 DateTime
UpdateUser 更新者 String 50
UpdateUserID 更新人 Int32 N
UpdateTime 更新时间 DateTime
UpdateIP 更新地址 String 50
Remark 备注 String 500

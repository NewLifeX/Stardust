应用配置（AppConfig）

名称 显示名 类型 长度 精度 主键 允许空 备注
Id 编号 Int32 AI N
ProjectId 项目 Int32 N 资源归属的团队
Category 类别 String 50
AppId 应用 Int32 N 对应StarApp
Name 名称 String 50 UQ
DisplayName 显示名 String 50
Enable 启用 Boolean N
Version 版本 Int32 N 应用正在使用的版本号，返回小于等于该版本的配置
NextVersion 下一版本 Int32 N 下一个将要发布的版本，发布后两者相同
PublishTime 定时发布 DateTime 在指定时间自动发布新版本
CanBeQuoted 可被依赖 Boolean N 打开后，才能被其它应用依赖
Quotes 依赖应用 String 50 所依赖应用的集合
IsGlobal 全局 Boolean N 该应用下的配置数据作为全局数据，请求任意应用配置都返回
Readonly 只读 Boolean N 只读应用，不支持客户端上传配置数据，可用于保护数据避免被错误修改
EnableWorkerId 雪花标识 Boolean N 给应用端分配唯一工作站标识，用于生成雪花Id，随着使用递增
EnableApollo 阿波罗 Boolean N
ApolloMetaServer 阿波罗地址 String 50
ApolloAppId 阿波罗账号 String 50
ApolloNameSpace 阿波罗命名空间 String 50 默认application，也可以填依赖的公共命名空间，但建议为公共命名空间建立应用依赖。
UsedKeys 已使用 String 2000 用过的配置项
MissedKeys 缺失键 String 2000 没有读取到的配置项
CreateUserID 创建者 Int32 N
CreateTime 创建时间 DateTime
CreateIP 创建地址 String 50
UpdateUserID 更新者 Int32 N
UpdateTime 更新时间 DateTime
UpdateIP 更新地址 String 50
Remark 备注 String 500


应用规则（AppRule）

名称 显示名 类型 长度 精度 主键 允许空 备注
Id 编号 Int32 AI N
Rule 规则 String 500 分号分段，逗号间隔多项匹配值，比如IP=10.*,172.*;LocalIP=192.*
Result 结果 String 50 比如Scope=dev
Priority 优先级 Int32 N 数字越大优先级越高
Enable 启用 Boolean N
CreateUserID 创建者 Int32 N
CreateTime 创建时间 DateTime
CreateIP 创建地址 String 50
UpdateUserID 更新者 Int32 N
UpdateTime 更新时间 DateTime
UpdateIP 更新地址 String 50
Remark 备注 String 500


应用依赖（AppQuote）

名称 显示名 类型 长度 精度 主键 允许空 备注
Id 编号 Int32 AI N
AppId 应用 Int32 N 原始应用，该应用依赖别人
TargetAppId 目标应用 Int32 N 被原始应用依赖的应用
CreateUserID 创建者 Int32 N
CreateTime 创建时间 DateTime
CreateIP 创建地址 String 50


配置数据（ConfigData）

名称 显示名 类型 长度 精度 主键 允许空 备注
Id 编号 Int32 AI N
AppId 应用 Int32 N
Key 名称 String 50 N 下划线开头表示仅用于内嵌，不能返回给客户端；多级名称用冒号分隔
Scope 作用域 String 50
Value 数值 String 2000 正在使用的值，支持内嵌 ${key@app:scope}
Version 版本 Int32 N 当前版本号，每次修改都是应用版本加一
Enable 启用 Boolean N
NewValue 期望值 String 2000 已被修改，尚未发布的值，支持内嵌 ${key@app:scope}
NewVersion 新版本 Int32 N 下一个将要发布的版本，发布后两者相同
NewStatus 新状态 String 50 启用/禁用/删除
CreateUserID 创建者 Int32 N
CreateTime 创建时间 DateTime
CreateIP 创建地址 String 50
UpdateUserID 更新者 Int32 N
UpdateTime 更新时间 DateTime
UpdateIP 更新地址 String 50
Remark 备注 String 500


配置历史（ConfigHistory）

名称 显示名 类型 长度 精度 主键 允许空 备注
Id 编号 Int32 AI N
AppId 应用 Int32 N
Action 操作 String 50 N
Success 成功 Boolean N
Remark 内容 String 2000
TraceId 追踪 String 50 最新一次查看采样，可用于关联多个片段，建立依赖关系，随线程上下文、Http、Rpc传递
CreateUserID 创建者 Int32 N
CreateTime 创建时间 DateTime
CreateIP 创建地址 String 50

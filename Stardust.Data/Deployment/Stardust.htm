应用部署（AppDeploy）

名称 显示名 类型 长度 精度 主键 允许空 备注
Id 编号 Int32 AI N
ProjectId 项目 Int32 N 资源归属的团队
AppId 应用 Int32 N 对应StarApp
Category 类别 String 50
Name 名称 String 50 UQ 应用名
Enable 启用 Boolean N
Nodes 节点 Int32 N 该应用部署集所拥有的节点数
Version 版本 String 50 应用正在使用的版本号
MultiVersion 多版本 Boolean N 支持多运行时版本，此时只认可部署版本中符合运行时要求的最新可用版本
AutoPublish 自动发布 Boolean N 应用版本后自动发布到启用节点，加快发布速度
PackageName 包名 String 50 用于判断上传包名是否正确，避免错误上传其它应用包，支持*模糊匹配
Port 监听端口 Int32 N 应用自身监听的端口，如果是dotnet应用会增加urls参数
Urls 服务地址 String 50 对外提供服务的域名端口地址，自动生成nginx配置，如https://sso.newlifex.com
Repository 代码库 String 200 下载代码的位置
Branch 分支 String 50 默认master
ProjectPath 项目路径 String 50 需要编译的项目路径，相对于代码库根目录
ProjectKind 项目类型 Int32 N 默认dotnet
BuildArgs 编译参数 String 200 编译项目时所需参数
PackageFilters 打包过滤器 String 50 需要打包哪些文件，支持通配符，多项分号隔开
FileName 文件 String 50 应用启动文件，可直接使用zip包
Arguments 参数 String 500 启动应用的参数
WorkingDirectory 工作目录 String 200 应用根目录
UserName 用户名 String 50 以该用户执行应用
Environments 环境变量 String 500 启动应用前设置的环境变量
MaxMemory 最大内存 Int32 N 单位M，超过上限时自动重启应用，默认0不限制
Priority 优先级 Int32 N 表示应用程序中任务或操作的优先级级别
Mode 工作模式 Int32 N 0默认exe/zip；1仅解压；2解压后运行；3仅运行一次；4多实例exe/zip
AutoStop 自动停止 Boolean N 随着宿主的退出，同时停止该应用进程
ReloadOnChange 检测变动 Boolean N 当文件发生改变时，自动重启应用
CreateUserId 创建者 Int32 N
CreateTime 创建时间 DateTime
CreateIP 创建地址 String 50
UpdateUserId 更新者 Int32 N
UpdateTime 更新时间 DateTime
UpdateIP 更新地址 String 50
Remark 备注 String 500

Priority 优先级 Int32 N 表示应用程序中任务或操作的优先级级别

Delay 延迟 Int32 N 批量发布时，需要延迟执行的时间，用于滚动发布，单位秒
ProcessId 进程 Int32 N

ProcessUser 进程用户 String 50 启动该进程的用户名
StartTime 进程时间 DateTime
Version 版本 String 50 客户端
Compile 编译时间 DateTime 客户端
Listens 监听端口 String 200 网络端口监听信息
LastActive 最后活跃 DateTime 最后一次上报心跳的时间
LastUpload 最后上传 DateTime 最后一次上传客户端配置的时间
TraceId 追踪 String 50 最新一次查看采样，可用于关联多个片段，建立依赖关系，随线程上下文、Http、Rpc传递
CreateUserId 创建人 Int32 N
CreateTime 创建时间 DateTime
CreateIP 创建地址 String 50
UpdateUserId 更新者 Int32 N
UpdateTime 更新时间 DateTime
UpdateIP 更新地址 String 50
Remark 备注 String 500
